## 🐛 Troubleshooting

### Frontend not loading?
- Check Vercel deployment logs
- Verify index.html was deployed
- Clear browser cache

### Backend 502/503?
- Wait 30 seconds (cold start)
- Check Render logs
- Verify service is "Live" in dashboard

### CORS errors?
- Should be fixed (CORS enabled in API)
- If issue persists, check browser console

### API not connecting?
- Verify URL in index.html: `https://air-draw-pro.onrender.com`
- Test backend health endpoint directly
- Check browser network tab
